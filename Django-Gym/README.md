
Це Django-проєкт для спортзалу PowerGym: головна сторінка, список тренерів, обрані тренери та адміністрування даних.

- CRUD для тренерів
- Перегляд списку тренерів
- CRUD для класів/послуг
- Керування записами клієнтів

- Головна сторінка з розкладом та абонементами
- Перегляд тренерів
- Обране (favorites)
- Запис на тренування
- Сторінка класів
- Сторінка відгуків
- Галерея залу
- Реєстрація та вхід користувачів

1. Встановити залежності:
```bash
pip install -r requirements.txt
```

2. Застосувати міграції:
```bash
python manage.py migrate
```

3. Запустити сервер:
```bash
python manage.py runserver
```

- Реєстрація: `/register`
- Вхід: `/login`
- Адмін-доступ (CRUD): потрібен `is_staff=True`
